## Installation

Add this line to your application's Gemfile:

```ruby
gem 'content_engine',
  :git    => 'https://github.com/Fro0z1k/content_engine.git',
  :branch => 'master'
```
And then execute:

    $ bundle install

Or install it yourself as:

    $ gem install content_engine
